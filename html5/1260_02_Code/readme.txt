Code Example of Chapter 2, in HTML5 Game Development
--------------------------------------------------------

installing_jquery  : Linking the jQuery library and ensure it is loaded.
pingpong_setup		: setup the game elements of Ping Pong.
manipulate_objects : Changing CSS position of the DOM elements with jQuery.
pingpong_control   : Getting keyboard input and controlling the paddels in Ping-Pong game.
pingpong_control2  : Using game loop to process the input, supporting multiple inputs.
pingpong_moving_ball : Using game loop to move the ball and bounce away when hitting the edges.
pingpong_hitting_paddle: Basic collision detection between the ball and paddles in Ping-Pong game.
pingponig_scoring : Dynamically changing HTML text and content with jQuery.

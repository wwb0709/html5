Code Example of Chapter 6, in HTML5 Game Development
--------------------------------------------------------

audio_game_play_button		: Play button with sound effects.
audio_game_background		: Drawing the background of the music game.
audio_game_playback			: Playing the music with music dots moving.
audio_game_scene_transition	: Transiting from the menu scene to game scene.
audio_game_keyboard_input	: Controlling the music lines with J, K, L keys.
audio_game_adjust_volume	: Adjust the melody volume based on the player performance.
audio_game_recording		: Recording the music note data for the game.
audio_game_ending			: Handling the melody ending event.

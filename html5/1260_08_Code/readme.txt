Code Example of Chapter 8, in HTML5 Game Development
--------------------------------------------------------

websocket_connection_count	: Connect to server and server returns the total connection count.
websocket_send_message		: Sending the message to server and displaying the message in server terminal.
websocket_basic_chat		: Building a chat room to allow sending and receiving message to all users who connected to the server.
websocket_local_drawing		: Building a local-only drawing board with canvas.
websocket_sync_drawing		: Sync all drawing from all connected user via WebSockets. 
websocket_draw_n_guess		: Building a multiplayer draw-and-guess game.
websocket_decoration		: Decorating the game with CSS stylesheet.
